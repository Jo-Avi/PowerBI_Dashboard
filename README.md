📊 Sales & Profit Analysis Dashboard (Power BI)

🔷 Project Overview

This project analyzes sales performance using Power BI. It provides insights into revenue, profit, customer behavior, and regional performance.

🔷 Dataset

* Orders Data
* Order Details Data

🔷 Key KPIs

* Total Sales
* Total Profit
* Total Orders
* Total Quantity

🔷 Dashboard Features

* Sales Trend Over Time
* Sales by State & City
* Category & Sub-Category Analysis
* Payment Mode Distribution
* Profit Analysis

🔷 Tools Used

* Power BI
* Excel / CSV
* DAX (Data Analysis Expressions)

🔷 Insights

* Identified top-performing states and cities
* Found high sales but low profit categories
* Analyzed customer payment preferences

🔷 How to Use

1. Download the `.pbix` file
2. Open in Power BI Desktop
3. Explore interactive visuals
